## 核心技术实现
- 前端基于 React + TypeScript + Vite 开发，页面层采用`Ant Design`组件完成表单、表格、弹窗、分页和反馈提示，页面样式使用 Tailwind 原子类与css结合，降低自定义 CSS 依赖。
- 课程管理和学生管理都采用“查询条件 + 表格列表 + 弹窗表单”的交互结构，通过统一的查询状态对象管理分页、搜索和筛选，保证前后端查询参数同步。
- 接口调用集中封装在`client/src/api.ts`中，统一处理`token`注入、401 失效清理和错误消息抛出，减少页面层重复逻辑。
- 工作台图表使用SVG组件实现，避免额外引入图表库，同时根据服务端返回数据动态渲染柱状图、折线图和饼图。
- 学生管理相关后端接口补全了新增、编辑、删除及选课关联逻辑，并同步维护课程选课人数，确保工作台与课程列表统计口径一致。
